# Data Structures in Python

A compact Python library implementing core data structures with simple APIs and example usage.

## Features
- Stack (LIFO)
- Queue (circular buffer)
- Singly Linked List
- Binary Search Tree (insert, search, delete, traversals)
- Hash Map (separate chaining, resizing)

## Requirements
- Python 3.8+

## Run
```bash
python3 main.py
```
